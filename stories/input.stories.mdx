## Input输入框

基本表单组件

### 基础用法

可以使用`v-model`来实现数据的双向绑定

<Preview>
  <Story name="基础用法">
    {{
      components: { Input },
      template: `
            <form>
                <Input v-model="value" placeholder="请输入内容..."></Input>
            </form>
      `,
      data() {
        return {
          value: "",
        };
      },
    }}
  </Story>
</Preview>

### 尺寸

可以通过`size`参数来设置输入框的大小, 可选参数为`large`,`medium`,`small`, `extra-small`, 默认为 `medium`

<Preview>
  <Story name="size 尺寸">
    {{
      components: { Input },
      template: `
            <form style="display:flex; align-items: center">
                <Input v-model="value1" size="large" placeholder="请输入内容..." clearable class="mr10" ></Input>
                <Input v-model="value2"  placeholder="请输入内容..." clearable class="mr10" ></Input>
                <Input v-model="value3" size="small" placeholder="请输入内容..." clearable class="mr10"></Input>
                <Input v-model="value4" size="extra-small" placeholder="请输入内容..." clearable></Input>
            </form>
      `,
      data() {
        return {
          value1: "",
          value2: "",
          value3: "",
          value4: ""
        };
      },
    }}
  </Story>
</Preview>

### 清除输入内容

可通过设置`clearable`属性来开启清空按钮

<Preview>
  <Story name="clearable 清除输入">
    {{
      components: { Input },
      template: `
            <form style="display:flex">
                <Input v-model="value" placeholder="请输入内容..." clearable></Input>
            </form>
      `,
      data() {
        return {
          value: "",
        };
      },
    }}
  </Story>
</Preview>

### 禁用

使用`disabled`或者`readonly`开启禁用或只读状态
<Preview>
  <Story name="禁用">
    {{
      components: { Input },
      template: `
            <form style="display:flex">
                <Input v-model="value" placeholder="请输入内容..." disabled class="mr10"></Input>
                <Input v-model="value" placeholder="请输入内容..." readonly></Input>
            </form>
      `,
      data() {
        return {
          value: "",
        };
      },
    }}
  </Story>
</Preview> 


### 密码框

在`type="password"`的情况下, 可以开启`show-password`可以切换密码的显示和隐藏

<Preview>
  <Story name="密码框">
    {{
      components: { Input },
      template: `
            <form style="display:flex">
                <Input v-model="value" type="password" show-password placeholder="请输入密码..."></Input>
            </form>
      `,
      data() {
        return {
          value: "",
        };
      },
    }}
  </Story>
</Preview>

### 文本域

可通过设置属性`type`为`textarea`来使用文本域，用于多行输入。通过设置属性 rows 控制文本域默认显示的行数。
<Preview>
  <Story name="文本域">
    {{
      components: { Input },
      template: `
            <form style="display:flex">
                <Input v-model="value" type="textarea" placeholder="请输入..." class="mr10"></Input>
                <Input v-model="value" type="textarea" :rows="10" placeholder="请输入..."></Input>
            </form>
      `,
      data() {
        return {
          value: "",
        };
      },
    }}
  </Story>
</Preview>
