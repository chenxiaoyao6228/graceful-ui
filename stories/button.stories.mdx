## API

### Props 列表

| 参数         | 说明     | 类型    | 可选值      | 默认值  |
| ------------ | -------- | ------- | ----------- | ------- |
| type         | 按钮类型 | string  | primary     | primary |
| disabled     | 是否禁用 | boolean | -           | false   |
| round        | 是否圆角 | boolean | -           | false   |
| icon         | 按钮     | string  | -           | -       |
| iconPosition | 按钮     | string  | left, right | -       |

### Events 列表

| 参数  | 说明         | 类型 | 可选值 | 默认值 |
| ----- | ------------ | ---- | ------ | ------ |
| click | 鼠标点击事件 | -    | -      | -      |
