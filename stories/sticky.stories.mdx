# Sticky 吸顶

滚动吸顶效果

### 引入组件

```js
import { Sticky } from 'graceful-ui';
```

<style>{`
  .demo{
    padding: 0 300px;

### 基本使用
使用`distance`参数来设置吸顶元素距离顶部的触发距离, `distance`默认为`0`
<Preview>
  <Story name="基本使用">
    {{
      components: { Sticky , Button},
      template: `
        <div class="demo">
          <header class="header">
            这里是顶部内容
          </header>
          <Sticky distance="100">
            <Button type="primary" long>吸顶元素, distance为100时触发</Button>
          </Sticky>
          <Footer class="footer">
            这里是底部内容
          </Footer>
        </div>
      `,
    }}
  </Story>
</Preview>
