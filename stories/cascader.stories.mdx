# Cascader 级联选择

从一组相关联的数据集合中进行选择，常用于省市区、公司级层、事务分类等。

### 引入组件

```js
import { Cascader } from 'graceful-ui';
```

<style>{`
   .demo {
     height: 200px;
   }
`}</style>

### 默认数据

指定 value 默认值，组件会在初始化时选定数据。

<Preview>
  <Story name="默认数据">
    {{
      components: { Cascader },
      template: `
        <div class="demo">
            <cascader :source="dataSource" v-model="value"></cascader>
        </div>
      `,
      data() {
        return {
          dataSource,
          value: ['广东', '广州', '天河'],
        };
      },
    }}
  </Story>
</Preview>
