# Uploader 上传

### 引入组件

```js
import { Uploader } from 'graceful-ui';
```

### 基本使用

<Preview>
  <Story name="基本使用">
    {{
      components: { Uploader, Button},
      template: `
        <div>
          <Uploader action="https://jsonplaceholder.typicode.com/posts/">
            <Button icon="upload">上传文件</Button>
          </Uploader>
        </div>
      `,
    }}
  </Story>
</Preview>

### 带缩略图

设置`listType`为`picture`开启缩略图模式

<Preview>
  <Story name="带缩略图">
    {{
      components: { Uploader, Button},
      template: `
        <div>
          <Uploader action="https://jsonplaceholder.typicode.com/posts/" listType="picture">
            <Button icon="upload">上传文件</Button>
          </Uploader>
        </div>
      `,
    }}
  </Story>
</Preview>
